Bu bir [Next.js](https://nextjs.org) projesidir.

## Başlarken

Öncelikle, geliştirme sunucusunu çalıştır:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

[http://localhost:3000](http://localhost:3000) adresine git.
